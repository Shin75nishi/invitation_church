# 招待キリスト教会 (Invitation Christian Church) Website

This is a simple HTML/CSS/JavaScript website for a fictional church called "招待キリスト教会" (Invitation Christian Church). The website is designed to be clean, modern, and responsive, providing essential information about the church, its services, events, and more.

## Project Structure

```
invitation-church/
├── css/
│   └── style.css          # Main stylesheet
├── js/
│   └── main.js            # JavaScript functionality
├── images/                # Directory for all images
├── index.html             # Homepage
├── about.html             # About page
└── README.md              # This file
```

## Features

- Responsive design that works on mobile, tablet, and desktop
- Modern, clean aesthetic with a focus on readability
- Mobile-friendly navigation menu
- Sections for church information, service times, upcoming events, and more
- Smooth scrolling and other interactive elements

## Pages

1. **Home (index.html)**: Main landing page with welcome message, service times, upcoming events, and latest sermon
2. **About (about.html)**: Information about the church, its vision, mission, staff, and beliefs
3. **Worship**: Information about worship services (to be implemented)
4. **Events**: Calendar and details of upcoming events (to be implemented)
5. **Sermons**: Archive of past sermons (to be implemented)
6. **Access**: Directions and map to the church (to be implemented)
7. **Contact**: Contact form and information (to be implemented)

## How to Use

1. Download or clone this repository
2. Open any of the HTML files in a web browser to view the website
3. Customize the content, images, and styling to fit your needs

## Customization

### Content

Edit the HTML files to update the text content, links, and structure of the pages.

### Styling

The main stylesheet is located at `css/style.css`. You can modify colors, fonts, spacing, and other visual elements here.

Key variables that can be easily changed:

```css
:root {
    --primary-color: #3b5998;
    --secondary-color: #8b9dc3;
    --accent-color: #dfe3ee;
    --text-color: #333;
    --light-text: #fff;
    --dark-bg: #1e2b4a;
    --light-bg: #f7f7f7;
    --font-sans: 'Noto Sans JP', sans-serif;
    --font-serif: 'Noto Serif JP', serif;
}
```

### Images

Replace the placeholder images in the `images/` directory with your own images. Make sure to maintain the same filenames or update the references in the HTML files.
